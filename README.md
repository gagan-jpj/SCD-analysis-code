# Data

This folder is intentionally empty.

## Why is there no data here?

All data used in this project is drawn from the
**NIH All of Us Research Program Controlled Tier Dataset v8**
and lives exclusively within the All of Us secure research enclave.

Participant-level data **cannot** be downloaded, stored locally, or shared
publicly under the All of Us Data Use Agreement.

## How to access the data

1. Apply for Researcher Workbench access at:
   https://workbench.researchallofus.org/

2. Complete the required training modules and data use agreement

3. Once approved, run the notebooks in this repository from inside
   your All of Us Workspace — the BigQuery queries will populate
   all required CSV files automatically
